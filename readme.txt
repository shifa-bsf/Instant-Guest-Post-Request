== Description ==

Instant Guest Post Request is a powerful WordPress plugin that allows visitors to submit guest posts directly from your website without requiring them to register or log in. This plugin is perfect for content-heavy blogs, community sites, and influencer outreach teams looking to streamline their content submission process.

= Key Features =

* **Frontend Submission Form** - Easy-to-use form accessible via shortcode `[guest_post_form]`
* **Rich Text Editor** - Full WordPress editor for post content
* **Featured Image Upload** - Allow users to upload featured images for their posts
* **Author Information** - Collect author name, email, and bio
* **Admin Moderation** - Review submissions before publishing
* **Email Notifications** - Get notified of new submissions with approve/reject links
* **Customizable Email Templates** - Personalize approval and rejection emails
* **Submission Limits** - Control how many posts can be submitted from the same IP
* **Category Selection** - Set default category for submissions

= Target Audience =

* Content-heavy blogs looking to expand their contributor base
* Community websites seeking user-generated content
* Influencer outreach teams managing guest post submissions
* Any WordPress site wanting to accept content from non-registered users

== Installation ==

1. Upload the `instant-guest-post-request` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Guest Post Settings to configure the plugin
4. Add the shortcode `[guest_post_form]` to any page where you want the submission form to appear

== Frequently Asked Questions ==

= Do users need to register to submit posts? =

No, users can submit posts without registering or logging in.

= How do I moderate submissions? =

All submissions are saved as pending posts. You can review them in the WordPress admin area under Posts, or use the approve/reject links in the notification emails.

= Can I customize the submission form? =

The form includes all essential fields. For advanced customization, you can modify the template files in the plugin.

= How do I limit spam submissions? =

The plugin includes IP-based submission limits to prevent spam. You can set the maximum number of submissions allowed per IP address in a 24-hour period.
